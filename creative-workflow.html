Creative Factory Intake

Step-by-step triage, platforms, and routing paths

Triage Mechanism

Dust Bot Threaded Assessment

When a requester posts in #creative-requests, the Dust Bot detects the message and responds directly in a Slack thread. It asks a dynamic series of questions to assess project size, visibility, and required disciplines before automatically routing the request to one of the 4 paths below.

Terminology & Rules
MODE A Strategic

Multi-disciplinary work planned in Panorama. Requires full CFT (Creative Factory + Brand Design).

MODE B Craft / Reactive

High-value, single-discipline requests. Assigned directly to specialists.

✅ Rule of thumb: The more the work needs a new hook (or sits on a flagship surface), the more it shifts toward Mode A. If it’s mostly execution with a polish pass, it’s Mode B.
